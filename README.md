Лабораторная работа №15: Разработка простого веб-сервиса на Flask

Цели: Освоить Flask, создать веб-сервер, разработать REST-сервис, изучить структуру проекта и маршрутизацию.

Задачи:

1. Создать веб-сервис на Flask с маршрутами
2. Реализовать обработку данных (URL-параметры, формы, JSON)
3. Подготовить отчёт и репозиторий с требуемой структурой

Общее задание (task_1): Создать базовый веб-сервис с маршрутами:

· / — приветствие
· /hello/<name> — персональное приветствие
· /square/<int:number> — квадрат числа
  Структура: task_1/app/init.py, routes.py, main.py, README.md

Индивидуальное задание (task_2) - Вариант 5:

1. GET /calc — принимает параметры a, b, op (+, -, *, /), возвращает результат в JSON
2. GET /status — возвращает JSON: {"status": "running", "service": "Flask App"}
