# Индивидуальное задание (Вариант 5)

## Реализованные маршруты

1. **GET /calc**
    Параметры (query string):):**
        *   a (число) — первый операнд.
        *   b (число) — второй операнд.
        *   op (строка) — оператор (+, -, *, /).
    Ответ (JSON):):** {"a": 5, "b": 3, "op": "+", "result": 8} или {"error": "..."}.

2. **GET /status**
Ответ (JSON):JSON):** {"status": "running", "service": "Flask App"}
