# Задание 1: Создание простейшего веб‑сервиса на Flask

## Цель
Освоить основы работы с микрофреймворком Flask: создать базовый веб‑сервер, обрабатывать запросы и формировать ответы.

## Реализованные маршруты
1. **GET /** — возвращает приветствие.
2. **GET /hello/<name>** — возвращает персональное приветствие.
3. **GET /square/<int:number>** — возвращает квадрат переданного числа.

## Запуск приложения
1. Установите зависимости: pip install flask
2. Запустите сервер: python main.py
3. Откройте в браузере: http://127.0.0.1:5000
